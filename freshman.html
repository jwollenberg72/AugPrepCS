Freshman will learn a block based language during the first semester to learn programming fundamentals like algorithms, input and output, conditionals, loops and functions. By the second semester, we will move onto Python, which is a powerful and easy text based language. We will go over the same programming fundamentals with Python that we did with our block based language. Students will also make music using Earsketch, which is a Python based program. No matter what your starting point is in programming, you will be challenged with fun, useful programs to write.
